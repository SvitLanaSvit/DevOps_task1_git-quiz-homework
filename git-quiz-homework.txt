Скрін з результатами проходження Git Quiz


Git merge types — це типи злиття гілок, наприклад:

    - Fast-forward merge

    - Three-way merge

    - Squash merge

    - Rebase (не merge, але часто вживається замість нього)

Git hooks — це скрипти, які Git виконує автоматично на певні події (наприклад, перед комітом або пушем). Вони зберігаються в .git/hooks.
